ES6 範疇 (scope)

區塊範疇 (block scoping)

使用 let 以及 const 可以創建繫結至 (bound to) 任何區塊的宣告，叫做區塊範疇 (block scoping)。

Temporal Dead Zone (TDZ，暫時死亡區)

過早存取由 let 所宣告的參考而產生的 ReferenceError 在技術上被稱作 Temporal Dead Zone (TDZ，暫時死亡區) 錯誤，你存取了已宣告但尚未初始化的變數。

let + for

常數 const 宣告

常數所限制的，並非那個值本身，而是加諸在那個值的變數指定(variable’s assignment)之上。

變數a實際上所持用的，並不是一個常數陣列(constant array)，而是對陣列的一個常數參考(constant reference)。該陣列本身可以自由變動。
