生命週期

生命週期方法 (Life Cycle)

componentWillUpdate()

在元件的 render() 方法再次被呼叫 (由於特性或狀態的改變) 之前被執行。

componentDidUpdate()

在 render() 方法完成且新的底層 DOM 變更發生之後被執行。

componentWillMount()

在節點被插入 DOM 之前被執行。

componentDidMount()

在節點被插入 DOM 之後被執行。

componentWillUnMount()

在元件從 DOM 移除之前被執行。

shouldComponentUpate(newProps, newState)

這個方法在 componentWillUpate() 之前被呼叫，並且可以回傳 false 來取消更新，這表示 render() 方法不會被呼叫，這在效能至關重要的領域中是相當有用的 - 當你認為沒有重要的變更，而且不需要渲染時。根據 newState 引數與既有 this.state 的比較，或者，單純知道這個元件是靜態、不會改變的，據以進行決策。
